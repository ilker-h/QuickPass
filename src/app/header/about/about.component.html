About QuickPass

This is QuickPass, an easy-to-use password manager that I created as a side project. Feel free to try it out, I hope you like it!

Built with:
  • Frontend: Angular 7 (TypeScript-based JavaScript framework)
  • Backend Hosting: Google Firebase's Realtime Database (NoSQL)
  • Layout Styling: Bootstap 4 (CSS framework)
  • Element Styling: Angular Material 7 (UI Component Library)
