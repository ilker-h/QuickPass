About uickPass

QuickPass is an easy-to-use password manager that I created as a side project. Feel free to try it out!

Built with:
  • Frontend: Angular 7 (TypeScript-based JavaScript framework)
  • Database Hosting: Google Firebase's Realtime Database (NoSQL)
  • Layout Styling: Bootstap 4 (CSS framework)
  • Element Styling: Angular Material 7 (UI Component Library)
  • Major Dependencies: @angular et al., Bootstrap, Core.js, Firebase, RxJS, tslib, Webpack, Zone.js
  • Other: HTML5 / CSS3, Material.io Icons, FontAwesome Icons
